# Lecture 1
## 与C语言对比
- 和C相似
    - 使用大括号和分号
    - 使用变量前要声明类型, 并永远不能改变(静态)(函数会快一点, 易读性增强)
    - 全部构建完毕以后(类型检查)才会运行程序
    - 函数也需要明确返回值
- 和C不同
    - Java非常注重面向对象
    - 大型函数和主函数都会放在一个class里
