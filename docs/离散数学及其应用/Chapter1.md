# **第一章:逻辑和证明**
## **1.1 命题逻辑**
- *命题(proposition)*
    - 称述性语言, 要么为真, 要么为假
- *命题变量(proposition variable)*
    - 用小写字母表示$p, q, r, s....$
- *真值(truth value)*
    - $T/F$
- *六种逻辑操作(logical operators)*
    - $\neg$ 否定(negation)
    - $\land$ 合取/且(conjunction)
    - $\lor$ 析取/或(disjunction)
    - $\oplus$ 异或(exclusive or)
    - $\rightarrow$ 蕴含(conditional)
    - $\leftrightarrow$ 双重蕴含(biconditional)
- *条件语句的等价形式*(记住这三个特殊的)
    - If $p$, then $q$
    - $p$ only if $q$
    - $q$ unless $\neg p$
